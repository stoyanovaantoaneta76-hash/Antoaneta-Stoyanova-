# Nordlys Model Engine - Core Library

This package contains the core logic used by Nordlys models. It is intended for internal development and research.

## Development

Install dependencies:

```bash
uv sync --package adaptive_router
```

Run tests:

```bash
uv run pytest adaptive_router/tests/
```

Formatting and checks:

```bash
uv run black adaptive_router/
uv run ruff check adaptive_router/
uv run mypy adaptive_router/
```
